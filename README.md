# Education-Access-System-Progress-Pathways
A group project "Educating Access System: Progress Pathways", aims to provide a comprehensive solution by creating a platform modeled on a learning management system.

Project Overview:
The Education Access System: Progress Pathways is a platform designed to make education accessible, inclusive, and flexible. It provides students with a structured way to access courses, programs, quizzes, and peer learning sessions, helping them develop skills and achieve their educational goals.

Key Components:
Detailed Use Case Diagram showing how students and admins interact with the system

Sequence Diagram representing step-by-step interactions within the system
State Diagram outlining the different states of user activities

Purpose:
This project demonstrates system planning, analysis, and design for creating an educational platform that supports inclusive and equitable learning opportunities.
